# Simple in memory database
## To run directly  
```
docker run -it puneetguptanitj/imdb
```
## Building jar/image locally
Pre-requisite javac

## Building
### jar
```
make build
```
### docker image
```
make docker
```
## Running
### jar
```
java -jar imdb.jar
```
### Container
```
docker run -it imdb
```
